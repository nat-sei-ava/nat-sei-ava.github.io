Udacity Reflection #2

What is grid based design?

Grid base design breaks down the design of a webpage into columns and rows. Making it easier to apply styles with CSS.

Whats all the hype about responsive webpages?

Responsive webpages are all the hype because of the proliferation of smart phones, tablets, iPads, laptops and all the other devices we use in our daily lives. Having pages that can repsond accordingly to whatever device we are using is essentially what responsive webpage design is about.

What is semantic structure?

Semantic structure are a set of rules that govern a certain language. Just like words in the english language have rules so does HTML.
