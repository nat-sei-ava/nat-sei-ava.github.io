Design to Web

What a responsive site is, and why responsiveness is important.

A responsive site design adapts to whatever device being used. Essentially it is flexible and adjustable.

It is extremely important as it affects the users experience. Having consistency on various devices enhances the user experience.

What mobile first design is, and why it's important.

Mobile first design seeks to design first to fit the mobile screen. It's important as using a mobile is very different to using a desktop and many consumers primarily use their phones to access the internet. We use our thumbs on smart phones and on a desktop we would use a mouse so to be responsive it needs to be designed differently and with the end-user in mind.

Mobile first design goes hand in hand with responsive design however can be more restrictive and difficult as it is the smallest screen to design for.

What frameworks are, and their pros and cons

A framework is like a template or package of standard code of HTML,CSS, and JS. It serves as a packaged base to start your website. I used an analogy of a cake to describe HTML,CSS and DOM. In this case a framework would be like a pre-packaged baking mix. No need to bake it from scratch you just add a few things and you have a cake or a website.

Pros of using a framework is they help you code faster and easier. The obvious con is it may not be better or the best it can be. Again the analogy of a cake, Everyone knows the best cakes are never made from packet mixes nonetheless they are still good. In my experience cakes made from scratch are the best but require more time, effort and skill.

What a wireframe is and why we use it.

A wireframe is a visual blueprint of webpage elements, features and navigation in a webpage. Wireframes help to plan out content and the user experience first.

The aspects of your wireframes you found difficult to implement, and why.

I found it difficult to visualise features like fonts and design elements because a wireframe is just a layout of the site. I think wireframing is difficult for designers as it doesn't include any design elements.
